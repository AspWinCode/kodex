# Codex — Universe Bible

---

## Назначение

Фиксирует вымышленный мир, в который упакована учебная программа, и жёсткие языковые правила, чтобы содержание оставалось последовательным при добавлении нового контента. Согласовано с пользователем в начале проекта (см. память сессии `codex-player-product-model`).

---

## 1. Предпосылка

Игрок — агент детективного агентства **«Кодэкс»** (KODEX), которое расследует преступления с помощью анализа данных и написания кода. Противник, упоминаемый на фоне (не как игровой уровень, а как атмосферный элемент) — сеть **«Хаос»**.

Это **не LMS**. В интерфейсе нет слов «урок», «тест», «оценка», «домашнее задание». Вместо них: дело, брифинг, материалы, улики, верстак, проверка, отчёт, допуск, ранг, кредиты, куратор.

## 3. Терминология (обязательна к соблюдению в новом контенте)

| Игровой термин | Означает |
|---|---|
| Дело (case) | единица учебного контента |
| Улика (evidence) | тест-кейс, проверяемый авточекером |
| Материал | учебный текст/пример кода |
| Верстак | редактор кода |
| Проверка улик | запуск авточекера |
| Отчёт | выбор версии произошедшего + награды |
| Допуск (rank) | уровень доступа к делам по сложности |
| Картотека | список дел |
| Полигон | тренировочные упражнения вне сюжета |
| Кабинет расследования | пространство одного дела (брифинг→материалы→верстак→проверка→отчёт) |

Любое системное сообщение — включая ошибки авточекера и офлайн-статус — формулируется в этой терминологии, не в терминах программирования или LMS напрямую (например: «обработчик не запустился», а не «SyntaxError на верстаке»).

## 4. Визуальный стиль

Тёмная тема: неоново-зелёный `#00ffab` + циан `#35c7ff` на почти чёрном фоне `#05070a`. JetBrains Mono для метаданных/заголовков, Golos Text для основного текста. Кнопки — неоновая обводка, заливка на hover. Онбординг кинематографичен: терминальная печать → письмо Виктора Кодэкса → выбор позывного → «досье создано».

Форм логина/регистрации нет — вход только через SSO (токен во фрагменте URL) либо dev-сессия в прототипе.

---

## 5. Экономика

Кредиты и репутация начисляются за раскрытые дела. Ранг (I–IV) растёт с репутацией и открывает доступ к засекреченным делам. Снабжение (Полигон/Личный терминал) — расходуемые и постоянные предметы за кредиты. Попытки на верстаке ограничены (`MAX_ATTEMPTS`), исчерпание — кулдаун, не провал навсегда.
