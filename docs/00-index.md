## Текущее состояние реализации

Пройдены **Foundation** (монорепозиторий, CI), **Core Platform** (формализация Engine поверх `apps/player`: именованные состояния, журнал игровых событий), большая часть **MVP**/**v0.2** (10 играбельных дел, диалоги, подсказки, Джарвисмен) и **v0.3, минимальный Studio** (`apps/studio`).

Поверх v0.3 сделан первый настоящий шаг Publishing: `services/content-api` — единственный сервис проекта (без БД, без фреймворков), хранящий Studio-правки в общем JSON-файле на сервере вместо localStorage браузера. Правка методиста теперь видна Player с любого устройства и переживает перезагрузку — это закрывает главный разрыв перед **v0.4 (AI Generation)**, для которой нужен настоящий, а не локальный цикл черновик → публикация. Версии, рецензия и совместная работа над одним делом (Studio Architecture, документ 09) — всё ещё впереди. Полноценный AI Generation и Analytics, описанные в документах 07, 15, пока существуют только как архитектурная спецификация.

Развёрнуто на VPS: `kodex.tirskix.space` (см. память сессии `vps-deployment` — сервер, репозиторий в `/opt/codex`, nginx проксирует на `content-api`).
