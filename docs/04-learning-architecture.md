## Назначение

Как платформа реально учит программированию на Python, поверх маскировки под детективное агентство (см. `05-universe-bible.md`). Это описание того, что фактически реализовано в `packages/game-data/data.js` и подтверждено `scripts/check.mjs` + сквозными прогонами через `services/python-runner`, а не аспирационная программа.

## 3. Педагогический цикл одного дела

Брифинг (мотивация, кто и зачем просит) → Материалы (теория как «досье»: `справка`/`форензика`/`запись`, всегда с примером кода) → Верстак (написание решения) → Проверка улик (авточекер, реальный запуск в Python Runner) → Отчёт (выбор версии произошедшего — мини-синтез, не тест на память) → Финал (реакция куратора).

Материал **не абстрактная документация** — он написан под конкретную задачу дела и обычно содержит готовый фрагмент кода, который решение почти напрямую использует. Задача агента — собрать решение из показанных приёмов, а не изобрести их с нуля.

## 4. Подсказки (Джарвисмен)

Три уровня на дело:
1. Направление — какой материал перечитать.
2. Метод — конкретная конструкция (без полного кода).
3. Подход целиком — полный код решения текстом (не вставляется в редактор автоматически — агент переносит вручную).

Экономика: наводки платные (кредиты) либо бесплатные при `hintTokens`; при `failStreak >= 2` Джарвисмен сам предлагает уровень I бесплатно (см. `apps/player/js/screens.js`).

---

## 5. Оценивание — только через реальное исполнение

Никаких эвристик по тексту решения (не проверяется «использован ли цикл» и т.п.). Единственный критерий — прохождение тестов `evidence[].tests` при реальном запуске кода в изолированном Python Runner (`services/python-runner/harness.py`). Это касается и «визуальных» дел (turtle/matplotlib) — MiniTurtle/MiniChart возвращают данные, которые сравниваются как обычный результат функции, а не эвристикой по «нарисованной картинке».

## 6. Прогрессия сложности

`difficulty` (1–3) и `rank` (допуск I–III) — два независимых измерения: `difficulty` — сложность самой задачи, `rank` — на каком этапе учебного плана дело открывается картотекой. Rank растёт с репутацией (см. `06-domain-model.md`).
